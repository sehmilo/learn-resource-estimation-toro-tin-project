Resource estimation is the discipline of answering one deceptively simple question with enough rigour that a bank will lend against the answer: how much metal is in the ground, at what grade, and how sure are we? This course builds that capability from first principles.

Why this course is built around Toro

You are advancing licence ANRML26-137 (39.08 km², Toro LGA, Bauchi State) through the Nigerian Younger Granites tin province. The project is unusual and instructive because it carries two fundamentally different orebody types that demand two different estimation philosophies:

The Alluvial Target

Placer Cassiterite-bearing wash gravels in the Bishiwai, Yadabongo and Kadade drainages. A volume-grade problem, estimated and reported in g/m³, sampled by pitting and bulk washing.

The Primary Target

Hard rock Greisen and pegmatite cassiterite-columbite in the topaz-granite cupola (LV 029). A grade-tonnage problem, estimated in % Sn, sampled by drilling and channel sampling.

The Benchmark

Every method is calibrated against real CRIRSCO-compliant reporting: Atlantic Tin's Achmmach project and the Alphamin Bisie NI 43-101 technical report sit in your reference library.

How to work through it

The modules are sequential: each one assumes the last. Every module follows the same rhythm — the theory derived in full, then applied immediately to real Toro data, then the same calculation expressed as Python you can run yourself.

The companion code

One script, python/toro_analysis.py, reproduces every figure and every number in this course from the raw project CSVs. The ordinary kriging system is hand-coded, not called from a library, so the mathematics in Module 5 maps line-for-line onto the code. Run it with python3 toro_analysis.py.

A note on scope and honesty

The Toro reconnaissance dataset is deliberately small. That is not a weakness of the course — it is the lesson. You will see exactly why 26 pits cannot yet support a Mineral Resource, what the data can and cannot defensibly say, and what the next drilling campaign must deliver to change that. Estimation is as much about quantifying ignorance as about declaring metal.
